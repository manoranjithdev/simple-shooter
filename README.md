# Simple Shooter UE5

## Features

- First-person shooter gameplay
- Enemy AI targeting and attack behavior
- Hit-scan weapon system
- Projectile and damage mechanics
- Health and death systems
- Dynamic HUD integration
- Custom GameMode and PlayerController logic
- Niagara muzzle flash and impact effects

---

## Tech Stack

- Unreal Engine 5
- C++
- Blueprints
- Git & GitHub

---

## Gameplay Systems

### Combat System
- Weapon firing mechanics
- Hit detection using traces
- Damage application system

### AI System
- Enemy targeting behavior
- Shooting intervals and attack logic
- Death handling

### UI/HUD
- Health bar updates
- Gameplay feedback systems
